# PRODIGY_DS_05

## Task 05 - Traffic Accident Data Analysis

Analyzed US traffic accident data to identify patterns related to accident severity, weather conditions, and time of day.

## Tasks Performed

* Loaded and cleaned accident dataset
* Analyzed accident severity distribution
* Visualized weather conditions during accidents
* Performed time-based accident analysis
* Identified traffic accident patterns and contributing factors

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab
* GitHub

## Key Insights

* Certain weather conditions contributed heavily to accidents
* Accident frequency varied significantly by time of day
* Different severity levels showed different distributions
* Traffic accident hotspots were influenced by environmental factors
